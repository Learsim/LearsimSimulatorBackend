API Endpoints

GET

  • /api/clients

    • Returns all the clients.

  • /api/config

    • Returns config.

  • /api/status

    • Returns status of the simulator and LearsimServer.

POST

  • /api/clients

    • Add a client.

PATCH

  • /api/clients

    • Change details of client.

  • /api/config

    • Change the config.

DELETE

  • /api/clients

    • Delete a client
